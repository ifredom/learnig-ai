# 什么是Tensor(张量)?

> 张量就是物理量，与张量相关的数学是对高维 (维数 ≥ 2) 的物理量进行 “量纲分析” 的一种工具。同一个物理量可以由不同的向量/矩阵表示出来，这是由线性空间（通常是流形的切空间）的对称性决定的。


## 想要了解张量（Tensor），首先需要对向量（Vector）有一个清晰的了解。

还有很多物理量用 向量 是没法表示的（后面会提到），向量其实是一个更广泛的表示方法的特例。
